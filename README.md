MythosLens
-
About:
-
MythosLens is a sample project I created to explore and deconstruct the meanings behind devotional lyrics and mythological stories.
-
I love listening to devotional music daily...... honestly once in a couple of months, but I often felt disconnected from the deep meanings behind the lyrics. I decided to build a platform that acts as a bridge. It’s currently in its early just initiate stage, but my plan  is to turn it into a global regional repository for mythological deconstruction.  I built this using a modern AI-native workflow (Cursor/Gemini/Vercel) to see how quickly I could go from idea to deployment.
-----------------------------------------------------------------------------------------------------------------------

Note:
-
This is currently a sample version. I have implemented a few examples (3 songs) to demonstrate the concept.

Features:
-
---> Lyric Deconstruction: A simple way to view the meaning behind traditional songs.

---> AI-Assisted Development: Built using Cursor and Gemini to bridge the gap between idea and code

---> Tech Stack used:  React, Next.js, v0.dev (UI), and Vercel (Deployment).
